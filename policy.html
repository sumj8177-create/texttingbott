v1.4 DMs, Member Profiles & Delete Messages Latest Jun 23, 2026
Major Feature
  • Member sidebar — shows all server members grouped by Online / Idle+DND / Offline with live status dots and avatars
  • DM view — click any member to open a full DM conversation with real-time incoming messages via SSE
  • Send DMs — type and send direct messages to any server member from inside the dashboard
  • Delete own messages — hover any message to reveal a trash icon; deletes both channel messages and DMs the bot sent
  • Back navigation — DM view has a clean back button that returns you to the server channel view
  • New backend routes: /members, /dm, /dm-events, /message DELETE
  • Enabled intents.members and intents.presences for live online status

v1.2 Custom Bots & Multi-Bot Sending Jun 20, 2026
Feature Fix
  • Custom bot support — add extra bot tokens to send messages as different bots
  • Bot selector pills — switch which bot sends a message before hitting Send

v1.1 Reply System, Notifications & SSE Live Feed Jun 15, 2026
Feature Fix Tweak
  • Reply system — click or hover any message to reply; reply bar shows context
  • Live SSE feed — messages appear in real-time without refreshing via Server-Sent Events
  • Notification badge — bell icon in header lights up when the bot is @mentioned or replied to
  • Message history — loads last 50 messages when you switch channels
  • Webhook logging for all bot events (connects, messages, sends)
  • Fixed SSE reconnect loop on connection drop
  • Escape key now correctly cancels an active reply
  • Textarea auto-resizes up to 120px as you type
  • Toast notifications for send success/failure/network errors
v1.0 Initial Release — Bot Dashboard Jun 10, 2026
Major Feature
  • Token gate — paste your bot token to connect; stored in sessionStorage only
  • Server browser — lists all guilds the bot is in
  • Channel bar — pick any text channel as chips in the top bar
  • Message composer — send messages as the bot with Enter or the Send button
  • Status pill — shows bot online/offline state with a pulsing dot
  • Nuke endpoint /shutdown=nukeyay clears all sessions instantly
